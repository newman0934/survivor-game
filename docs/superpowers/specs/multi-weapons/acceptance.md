## 各武器行為正確
- [x] 魔杖：鎖定最近敵人、單發直線；高等級可鎖定最近 N 隻
- [x] 飛刀：朝玩家最後非零移動方向發射；多發時並排/扇形
- [x] 聖經：N 本以玩家為圓心環繞旋轉；碰到敵人扣血
- [x] 大蒜：玩家周圍圓形場域，範圍內敵人持續扣血（dmg×dt）

## 武器專屬升級（VS 風）
- [x] 每把武器最高 5 級
- [x] 升級套用該等級表預定的固定效果（數量/傷害/冷卻/半徑/轉速）
- [x] 升級卡面正確顯示等級資訊（如「飛刀 Lv2→Lv3」）

## 全域被動（乘區）
- [x] PlayerStats 改為乘區：damageMult / cooldownMult / projectileSpeedMult / areaMult + moveSpeed + pickupRadius
- [x] 全域被動同時影響所有武器的生效數值（生效值 = 等級值 × 乘區）
- [x] 既有 5 張升級卡改寫為乘區被動且行為正確

## 升級卡池
- [x] 候選動態依當前 World 狀態產生（持有/等級/上限）
- [x] 三類候選：解鎖新武器 / 升級既有武器 / 全域被動
- [x] 武器持有達上限 4 後，不再產生「解鎖新武器」候選
- [x] 武器達 Lv5 後，不再產生該武器的升級候選
- [x] 候選不足 3 張時以「補血」保底卡補滿至 3 張
- [x] 每次升級恰好提供 3 張不重複候選

## 邊界與錯誤處理
- [x] 場上無敵人時，鎖定型武器不開火、無例外、無除以零
- [x] 玩家從未移動時，飛刀朝預設方向（向右）發射
- [x] 一次拾取跨多個等級門檻時，每次升級都重抽當前合法候選
- [x] 套用未知升級 id 時安靜略過、不丟例外
- [x] 聖經對同一敵人在命中冷卻期間只扣血一次
